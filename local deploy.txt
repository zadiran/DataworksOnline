Минимальные требования:
- Python версии не ниже 3.4 
- СУБД PostgreSQL версии не ниже 9.1

1. На локальном сервере PostgreSQL необходимо создать супер-пользователя "Dataworks" с паролем "1234"
2. Создать на локальном сервере PostgreSQL базу данных "DataworksOnline"
3. Выполнить команду "pip install -r requirements.txt" в корневой папке для установки всех необходимых пакетов
4. Выполнить команду "python manage.py db upgrade" для обновления БД 
5. Выполнить "run.py" для запуска приложения
6. Приложение будет доступно по адресу "localhost:5000"
